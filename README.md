# Alpha

Alpha is a spotify like application built using Angular framework for the front-end, Node JS and MySQL as the back-end and database, it allows users to stream songs, rate various songs and artists as per their liking. Top songs and artists are displayed based on the average ratings of each registered user. 
New songs and artists can be added to the database.

### Key Features: ###

#### User Features (Front-end) ####
* Clean UI built using latest **Angular Framework** for the front-end and **MySQL DB** for the database.
* **REST APIs** are built using **Node JS** to communicate with the MySQL DB to perform operations.
* Users can register and login to *stream songs, rate various songs and artists.*
* **Top songs and artists** are displayed in the home page based on the average rating of each user registered with Alpha.
* Users can also **add songs by selecting existing artists or by adding new artists from the same page with the cover art** to the database.

#### Back-end Features ####
* Link to the [Alpha back-end repository](https://github.com/Abhishek-S-777/Alpha-Back-End)
* Link to [Alpha database design](https://github.com/Abhishek-S-777/Alpha-Front-End/files/9135170/Alpha-DB.Design.pdf)
* The back-end is built using **SQL**, **MySQL** has been chosen as the database as it supports large database and is very much reliable and fast.
* There are a total of 7 tables, which are normalized (2NF).
* Node JS is used to connect the front-end to the back-end using the REST APIs.
